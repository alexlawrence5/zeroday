# Zeroday

**zeroday** is an open-source hardware (ESP32) and software project focused on building privacy-first mobile phones.

The goal of this project is to create devices that prioritize **user privacy, transparency, and programmability**. Unlike most modern smartphones, zeroday devices are designed to be **fully inspectable, modifiable, and developer-friendly**.

## Key Ideas

* **Privacy-focused** – Minimal data collection and no hidden telemetry.
* **Open hardware** – Hardware designs are transparent and open for inspection.
* **Open software** – The operating system and software stack are open-source.
* **Programmable** – Developers can modify and program the system freely.

## Philosophy

Most modern smartphones are tightly controlled ecosystems. zeroday aims to explore an alternative: devices that users can fully understand, modify, and control.

This project experiments with new approaches to **secure, transparent, and user-controlled mobile computing**.

## Status

⚠️ This project is currently experimental and under development.

Features, hardware designs, and software components may change frequently.

## Goals

* Build privacy-oriented mobile hardware
* Develop open and transparent system software
* Allow deep programmability for developers and researchers
* Reduce dependence on proprietary mobile ecosystems
